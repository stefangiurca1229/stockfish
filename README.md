# Stockfish Chess UI

This repository now includes a browser-based Chess UI.

## Running the Chess UI

1. Open `chessui/index.html` in your web browser.
2. You will see a static chessboard. You can click a piece to select it, then click another square to move it.
3. No backend or chess engine integration is included. This is a static UI for demonstration and basic interaction only.

## Files
- `chessui/index.html`: Main HTML file
- `chessui/style.css`: CSS styles
- `chessui/app.js`: JavaScript logic

For further integration with a chess engine or backend, please specify additional requirements.
