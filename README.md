# Stockfish Home Screen

## Launching the Application

To start the application and access the new home screen:

```bash
python main.py
```

You will see a visually appealing home screen with two buttons:
- **Play vs Stockfish**: Start a game against the Stockfish engine.
- **Play vs Player**: Start a game against another player.

Click either button to proceed to the desired game mode. If a mode is not yet implemented, a placeholder or stub will be shown.

## Requirements
- Ensure all dependencies for the chosen UI framework are installed (see `requirements.txt` if applicable).

## Notes
- The home screen is the new main entry point for the application.
- Further enhancements to styling and accessibility may be added in future updates.
