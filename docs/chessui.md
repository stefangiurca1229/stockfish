# Chess UI Documentation

## Overview
The Chess UI is a browser-based frontend for displaying and interacting with a chessboard. It is implemented as a static web application with no backend or chess engine integration.

## Getting Started
1. Open `chessui/index.html` in your web browser.
2. You will see a chessboard. Click a piece to select it, then click a destination square to move it.
3. The UI is for demonstration and basic interaction only; it does not enforce chess rules or connect to a backend.

## File Structure
- `chessui/index.html`: Main HTML file for the UI
- `chessui/style.css`: Stylesheet for the UI
- `chessui/app.js`: JavaScript logic for rendering the board and handling user interaction

## Extending the UI
To integrate with a chess engine or backend, additional requirements and implementation are needed.
